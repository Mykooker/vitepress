# 1.web访问显示Network Error
a，在手机上关闭WiFi用流量访问看是否正常，如正常则是电脑网络问题
b，手机也无法访问，请提供你的手机号和访问地址私信客服处理

# 2.云机内显示网络异常或无网络
a，关闭云机代理，确保云机顶部4G信号旁无小钥匙图标
b，重启云机

# 3.一键新机是卡住不动
a，关闭云机代理，确保云机顶部4G信号旁无小钥匙图标
b，重启云机
